# Serverless Blog
A collection of lambda functions that constitute a blog service

### what it does:
- implements a set of simple lambda cruds for dynamoDB
- reacts to web hooks to create / delete blogs
- has a simple schema validation

### What it does not do:
- parses markdown
- adds metadata
